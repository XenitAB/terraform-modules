# Falco

Adds [`Falco`](https://github.com/falcosecurity/falco) to a Kubernetes clusters.  
The modules consists of two components, the main Falco driver and the sidekick which  
exports events to Datadog.

## Requirements

| Name | Version |
|------|---------|
| terraform | 0.13.5 |
| helm | 1.3.2 |
| kubernetes | 1.13.3 |

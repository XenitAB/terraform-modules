# Grafana Loki

This module is used to add [`loki`](https://github.com/grafana/loki) to Kubernetes clusters (tested with AKS).

## Details

This module will also add `minio` (S3 Gateway to Azure Storage Account), `fluent-bit` and `grafana`.

| Name | Version |
|------|---------|
| terraform | 0.14.7 |
| azurerm | 2.53.0 |

| kubernetes | 2.0.3 |
